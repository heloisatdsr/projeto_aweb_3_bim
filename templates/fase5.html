CSS

Como você já viu, o CSS é o código que você usa para dar estilo à sua página Web, permitindo a aplicação de estilos seletivamente a elementos em documentos HTML.

Mudar a cor da página

Deve-se utilizar a propriedade background-color aplicada ao seletor body. Defina o valor da propriedade com a cor desejada, usando nome de cor (ex: red), código hexadecimal (ex: #FF0000) ou RGB (ex: rgb(255, 0, 0)).
