Imagens

A tag <img> é usada para inserir imagens em uma página HTML. Ela precisa do atributo essencial src para indicar o arquivo da imagem e alt para descrever o conteúdo caso ela nao consiga.

Tabelas

As tabelas em HTML são criadas com a tag <table> e organizadas em linhas (<tr>) e colunas (<td>), servindo para exibir dados de forma estruturada, como planilhas e horários.

Formulários

Os formulários (<form>) permitem coletar dados do usuário, como nome ou e-mail. Utilizam campos como <input>,<textarea> e <button> para enviar as informações. 

        <form>
            <label>Enter Name:</label>
            <input type="text" name="name"><br><br>

            <label>Enter Password:</label>
            <input type="password" name="password"><br><br>

            <input type="submit" value="Submit">
        </form>

OBS: a tag <br> foi usada como quebra de linha.
